FLEX

This is the shorthand for flex-grow, flex-shrink
and flex-basis combined. The second and third
parameters (flex-shrink and flex-basis) are
optional. Defoult is 0 1 auto.


FLEX GROW

This defines the ability for a flex item to grow if
necessary. it accept a unitless value that
serves as a proportion. it distates what amount
of the available space insade the flex container
the item should take up.

FLEX SHRINK

This defines the for a flex items to shript
if necessary. Negativi numbers are invalid.
